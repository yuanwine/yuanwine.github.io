2.配置my.ini文件

  • 解压缩后如下图：

  • 在上图所示根目录下配置文件

    右键创建一个文本(.text)

    修改文本为my.ini(.ini文件格式与.text类似)

将下面的内容复制到my.ini中(将mysql的安装目录改成自己的下载路径)

不需要自己创建data存放路径MySQL会自己生成data文件

# 设置3306端口
port=3306
# 设置mysql的安装目录——文件路径
basedir=D:\MySQL\mysql-8.0.27-winx64
# 允许最大连接数
max_connections=200
# 允许连接失败的次数。
max_connect_errors=10
# 服务端使用的字符集默认为utf8mb4
character-set-server=utf8mb4
# 创建新表时将使用的默认存储引擎
default-storage-engine=INNODB
# 默认使用“mysql_native_password”插件认证
#mysql_native_password
default_authentication_plugin=mysql_native_password
[mysql]
# 设置mysql客户端默认字符集
default-character-set=utf8mb4
[client]
# 设置mysql客户端连接服务端时默认使用的端口
port=3306
default-character-set=utf8mb4

3.初始化数据库

  • 初始化mysql,在安装时避免权限问题以管理员身份运行cmd

      win+r输入cmd,ctrl+shift+enter以管理员身份运行

  • 否则在安装时可能会报错，导致安装失败

  • D: 转到你所安装的盘下

      cd bin的路径 转到bin下

  • 初始化mysql输入：

      mysqld --initialize --console

    记住初始化密码复制root@localhost:之后的密码( : 后有一个空格，不复制)忘记了就重新初始化.

    OX:XYkuUm4hj

4.安装mysql服务并启动+修改密码

  • 安装mysql服务

      mysqld --install mysql

  • 启动mysql服务

      net start mysql

  • 连接mysql

      mysql -uroot -p

  • 输入初始密码

  • 修改密码——

      ALTER USER 'root'@'localhost' IDENTIFIED BY '123456';

六.MySQL常见命令

  • 显示本机数据库：

      show databases;

  • 连接数据库(sys)：

      use <数据库>;

  use sys;

  • 显示数据库(sys)中的表：

      show tables;

…

  • 在(sys)库中查看mysql库中的表：

      show tables from mysql;

  • 当前选择的数据库：

      select database();

  • 创建一个(lhy)数据库：

      create database lhy;

  create database <数据库>;

  • 创建一个表(TheCollege)在库(lhy)中：

      create table <表名> ( <字段名1> <类型1> [,..<字段名n> <类型n>]);

  > create table TheCollege(
  -> id int,
  -> name varchar(20)
  -> );

  • 查看表(TheCollege)的结构：

      desc TheCollege;

  desc <表名>;

  • 查看表(TheCollege)中数据：

      select * from TheCollege;

  select * from <表名>;

  • 给表插入数据：

      insert into TheCollege (id,name) values(10590,'深圳大学');


  insert into <表名> (<字段名1>,...<字段名n>) values (值1,...值n);

    如果数据是字符型，必须使用单引号或者双引号括起来。

  • 修改表中数据：

      update TheCollege set name='华南理工' where id=10561;

  update 表名 set 字段=新值,… where 条件

  • 删除表中数据：

      delete from TheCollege where id=10561;

  delete from 表名 where 表达式;

  • 删除数据库(lhy)：

      drop database lhy;

  drop database <数据库>;
