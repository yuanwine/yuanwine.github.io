1.安装Node.js(js的运行环境)

  • Node.js官网[] (https://nodejs.org/en/)下载安装包，并进行安装。

  • 安装完成后，打开命令行cmd,输入node -v，npm -v查看版本号。

  • 查看系统变量中的环境变量

  • 查看用户变量中的环境变量：这里是npm包的下载路径，这里我已经更改了路径。

  • 配置npm的全局模块的存放路径以及缓存路径。(默认会安装到[C:\Users\用户名\APPData\Roaming\npm])

  • 在node.js的安装目录下新建俩个文件夹node_cache和node_global

  • 在命令行输入：

      1. npm config set prefix"你创建的node_global文件路径"
  2. npm config set cache "你创建的node_global文件路径"

  • 在重新配置npm的环境变量(上面修改了路径)：在系统变量中新增一个变量

  • 在用户变量中修改默认下载路径为你新建node_global的路径。

  • 安装个module测试下,命令行中输入：npm install express -g 完成后在你创建的node_global中的node_modules目录下就可以看到expres 文件夹和他的文件了。

  • 建议用管理员身份运行命令行

2.安装cnpm

  • npm因为是从国外服务器下载，可能出现异常，下载安装国内的服务器作为包源，以防不备。

  • 输入命令：

    npm install -g cnpm --registry=https://registry.npm.taobao.org

  • 完成后输入cnpm -v查看相关信息。

3.安装webpack(js应用程序的静态模块打包器)

  • 输入：

    cnpm install webpack -g

4.安装vue-cli(用来生成vue模板工具)

  • 输入：

    cnpm install vue-cli -g

5.新建一个Vue项目

  • 先进入想创建项目的盘中。

  • 再进入存放vue项目的文件夹中

  • 输入：

      vue init webpack firstvue

    firstvue为项目名。

  • 新建中要选择些选项，还有y/n。不懂的可以去翻译一下，了解了解。最后一步建议自己下载依赖。

  • cd firstvue 进入项目目录中。

  • 输入：

       cnpm install

    下载依赖

  • vue项目目录结构

  • 启动项目，到package.json中点击调试

  • 选择

      npm run dev

    运行完成后ctrl+单击,跳转页面。

6.使用Vue CLI脚手架快速创建项目

  • 有了脚手架，我们可以通过简单的命令就能快速生成 Demo 代码、构建本地测试环 境、编译和打包代码、发布到现网等功能。

  • Vue CLI 其实也是基于 Webpack 封装的便捷脚手架。

  • 安装脚手架：

       npm install -g @vue/cli

  • 脚手架生成vue项目，会自动安装依赖

      vue create vue-cli-twovue

    vue-cli-twovue为我项目名称，自行选择想要创建项目的盘和文件夹，操作同五。
